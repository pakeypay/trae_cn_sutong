# AGENTS.md

## 项目定位

这是复旦大学附属儿科医院技能培训中心的前端 Demo 项目。

当前阶段是客户展示 Demo，不接后端，不引入复杂服务，所有数据使用 mock data。

主要用户：
- 老师
- 学生
- 培训中心管理员

## 技术约束

- PC 端：Vue + Arco Design + Design_token覆盖样式
- 使用cdn导入vue，不新增后端依赖

## 设计方向

界面要专业、清爽、现代、可信。用户很喜欢飞书的ui风格。

主色使用复旦儿科医院技能培训中心蓝，#175898


不要使用：
- 克莱茵蓝
- 传统笨重后台风格

## 工作前必须阅读

- 定位修改都模块与影响范围：docs/project-brief.md
- 跟设计相关：docs/design-rules.md

## 工作规则

1. 不要全局大改。
2. 每次只改一个模块或一个页面。
3. 先说明计划，再修改代码。
4. 优先复用已有组件。
5. 如需新增组件，必须说明为什么，并且放到Design_token中。
6. 完成后说明修改了哪些文件、影响了哪些页面。
