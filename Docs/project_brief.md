# Project Brief

## 项目名称

复旦大学附属儿科医院技能培训中心教务与综合管理系统

## 项目阶段

当前是前端 Demo 展示阶段，不接后端。

目标是让客户直观看到系统的信息架构、业务流程、页面体验和视觉方向。

## 用户角色

### 老师

核心任务：
- 查看我的授课
- 查看今日/近期课程
- 上传课程资料
- 查看签到、问卷、评价
- 查看课程复盘

### 学生

核心任务：
- 查看我的课程
- 查看课程资料
- 签到
- 填写评价
- 查看证书

### 培训中心管理员(有总管理员，比如院长；也有分管理员：比如物资管理员，空间管理员，后台管理员)

核心任务：
- 管理课程
- 管理师资
- 管理学生
- 管理基地科室
- 管理场地预约
- 管理物资借用
- 查看统计与成果

## 当前重构目标

不是一次性重写项目，而是：

1. 统一视觉风格
2. 统一页面结构
3. 统一基础组件
4. 先做一个样板模块
5. 后续其他模块照样板复制
