# Design Rules
组件规则：使用Arco Design的组件库，但是用shared/design-tokens.css覆盖样式。

## 整体风格

关键词：
用户很喜欢飞书的ui风格。
- 医学教育感
- 不像笨重的传统后台模板
- 参考飞书网页版的设计，可以适当使用颜色。


## 颜色

主色使用复旦儿科医院技能培训中心蓝，#175898
参考飞书的颜色方案。

## icon
使用Lucide图标库。

## PC 端原则

PC 端重点是：

- 信息清晰
- 筛选高效
- 表格和卡片统一
- 操作路径明确
- 不堆砌太多按钮

常见页面结构：

1. 页面标题区
2. 数据概览区
3. 筛选区
4. 列表/表格区
5. 操作区

## 组件规则

优先统一这些组件：

- 页面容器
- 卡片
- 筛选栏
- 状态标签
- 课程卡片
- 统计卡片
- 空状态
- 操作按钮
- 移动端任务卡片

## 状态展示

不同状态使用轻量标签，不要使用过重的色块。

## 禁止事项

- 不要每个页面单独设计一种风格
- 不要过度使用大面积渐变
- 不要把老师端做成管理员后台
- 不要为了炫技增加无意义动效
- 不要新增复杂交互影响 Demo 稳定性
- 不接受任何位置出现emoji图标。
