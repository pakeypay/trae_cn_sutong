# 内容区 V2 第一阶段验收报告

## 1. 验收结论

第一阶段通过。课程开发已形成可复用样板，工作台、空间预约审批、空间资产管理和物资管理已迁入 V2 内容区结构。物资六个业务入口均可显示，不再出现管理员端内容区被清空的白屏问题。

## 2. 已建立的公共契约

- `app-page-header`：一级页标题、说明和主操作。
- `app-task-header`：返回、任务标题、状态、保存与发布。
- `app-view-tabs`：对象范围和同级视图。
- `app-control-bar`：搜索、筛选、重置、排序和视图切换。
- `app-selection-bar`：选中数据后出现的批量操作。
- `app-context-bar`：风险、冲突和流程提示。
- `app-main-aside`：主内容与单一职责辅助栏。
- V2 规则只在 `.app-content-v2` 中生效，未破坏性覆盖旧页面。

## 3. 文字与间距

- 页面标题：20/28px。
- 编辑任务标题：22/30px。
- 区块标题：16/24px。
- 正文、表格、按钮和输入：14px。
- 元信息：13/20px。
- 12px 仅用于状态标签、编码角标和辅助说明。
- 课程列表、课程编辑器、空间页面已清除承载业务句子的 11px 文字。
- 页面左右 32px；1365px 以下收敛为 24px；区块间 24px；标准卡片 20px；紧凑卡片 16px。
